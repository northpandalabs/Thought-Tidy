🔒 Privacy & Security first

Before you start — here's exactly what happens with your data.

🔐

Key encrypted on your device

Your API key is encrypted with AES-256-GCM and stored only in your browser's local storage. It is never sent to us, never logged, never uploaded.

➡️

Direct connection — no middleman

When you run an action, your text goes directly from your browser → the AI provider's API. Thought Tidy has zero backend servers. There is nothing in between.

🚫

API calls don't train AI models

All three providers (Gemini, OpenAI, Claude) contractually guarantee that API traffic is not used to train or improve their AI models. This is different from using their free consumer apps.

📖

Open source — verify it yourself

Every line of code is public. You can read exactly what gets sent where. No telemetry, no analytics, no tracking of any kind.

View source on GitHub ↗

Important — Gemini free tier vs paid API

Google has two different tiers with different privacy terms:

Free (consumer) gemini.google.com / AI Studio unmetered May train on your data

Google's free consumer tier can use prompts to improve their models.

API with billing enabled ✓ Google AI Studio API key (pay-as-you-go) Data NOT used for training

Once billing is enabled on your Google Cloud account, the Google Cloud Terms of Service apply — your data is not used to train models. There is still a generous free quota every month.

The guide below shows you how to enable billing and get the privacy-safe API key.

🔒
API ≠ Consumer app

When you use the API directly (with your own key), you're bound by the developer/API terms of service — not the consumer product terms. Developer terms uniformly prohibit training on customer API data.

💳
Pay-per-use, not subscription

API billing is based on tokens (roughly 1 token = ¾ of a word). Fixing a 200-word paragraph costs fractions of a cent. You only pay for what you actually use.

🏠
Want zero cloud? Use Ollama

The Thought Tidy desktop app supports Ollama — run open-source AI models entirely on your own machine. No API key, no internet connection needed, no data ever leaves your computer. Available in the desktop app at GitHub ↗.

🌐
Open source, build on it

Thought Tidy is open source. You can fork it, modify it, self-host it, or integrate it into your own tools. No lock-in, no proprietary format. It's a wrapper — you own the API relationship directly.
