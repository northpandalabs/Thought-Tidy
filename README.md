## How it works

**Desktop app** (works from any application):

```
1  Copy your text                    Ctrl+C
2  Open Thought Tidy                 Ctrl+Shift+Space
3  Pick an action from the dropdown  Brain Dump → Clear Text
4  Click Run
5  Copy the result                   Ctrl+C → paste anywhere
```

**Browser extension** (on any webpage):

```
1  Select text on any page
2  Right-click → pick an action
3  Result appears in a modal — Replace or Copy
```

### Try it: Brain Dump example

You're writing a client email but the thoughts are coming out wrong. You paste this:

```
need to talk to client about the deadline moving it back 2 weeks cause the dev
had a issue with the auth thing and also the scope changed a bit and we want to
make sure the testing is done properly before we push anything live
```

You pick **Brain Dump → Clear Text**, click Run. You get:

```
I wanted to flag a couple of updates on our timeline.

Due to a technical issue with the authentication component and some scope
adjustments, we'll need to move the deadline back by two weeks. We want to
ensure thorough testing is completed before anything goes live.

Let me know if you'd like to discuss.
```

3 seconds. No ChatGPT tab. No copy-paste between windows. Just clean text where you need it.

## The "Sound Like Me" feature

Fill out your profile once — your name, your role, your writing style, personal context. Enable **Inject into every prompt** and every action now knows who it's writing for.

**Sound Like Me** rewrites text in your authentic voice: not generic AI, not formal corporate, just *you but clear*. The difference between "Dear Sir/Madam, I hope this email finds you well" and something that actually sounds like you wrote it.

**Profile fields:**
- **Your Name** — personalizes tone
- **Your Role / Job** — roofing contractor, developer, nurse, whatever
- **Writing Style** — "casual and direct, short sentences, use em-dashes"
- **Personal Context** — bio, who you write to, common topics

You can also load a profile from a URL (GitHub Gist, plain text file) with one click.

## Your Privacy & Data Safety

- **No Thought Tidy servers.** Your text travels from your browser or desktop app directly to the AI provider you chose. Nothing passes through our infrastructure.
- **No accounts, no sign-up.**
- **Local-only storage.** API keys, settings, writing profile — stored only on your machine, never uploaded.
- **Encrypted API keys.** Desktop: OS keychain (Windows DPAPI, macOS Keychain). Extension: AES-256-GCM.
- **Open to audit.** Full source on GitHub. Read every line before installing.

Security tests run on every release (`tests/security.test.js`) — XSS prevention, no key leakage, context isolation, no telemetry, sync token auth. If any fail, the build is blocked.

## Install locally

### Browser Extension

```bash
npm install
npm run build          # produces dist/chrome/ and dist/firefox/
```

**Chrome / Edge:**
1. `npm run build:chrome`
2. Open `chrome://extensions` → enable **Developer mode** → **Load unpacked** → select `dist/chrome/`
3. Click the toolbar icon → Settings → paste your API key → Save

**Firefox:**
1. `npm run build:firefox`
2. Open `about:debugging` → **This Firefox** → **Load Temporary Add-on…** → select any file inside `dist/firefox/`
3. Click the toolbar icon → Settings → paste your API key → Save

### Desktop App

```bash
cd desktop
npm install
npm start              # launches with DevTools open
```

First launch opens Settings automatically. Add an API key, click Test, close Settings. Then:
- `Ctrl+Shift+Space` opens the popup from anywhere
- Right-click the tray icon for quick clipboard actions

**Build installers:**
```bash
cd desktop
npm run dist:win       # → dist-build/*.exe  (Windows NSIS)
npm run dist:mac       # → dist-build/*.dmg  (macOS)
npm run dist:linux     # → dist-build/*.AppImage
```

The installed app checks for updates on launch and every 4 hours — downloads silently and prompts to restart.

## For developers

No bundler. No framework. Minimal build step (copies files, swaps manifest background per browser).

```
lib/
  api.js              ← AI provider calls (OpenAI, Anthropic, Gemini, Ollama)
  prompts.js          ← system prompts, profile injection, action definitions
  models.js           ← model fetching and caching per provider
  shared-popup.js     ← popup logic shared between extension and desktop
  shared-settings.js  ← settings page logic shared between extension and desktop
  license.js          ← Pro license verification (Gumroad API)
popup/                ← toolbar button popup
options/              ← extension settings page
desktop/
  main.js             ← Electron main process (tray, IPC, sync server)
  renderer/           ← desktop popup + settings window (HTML/CSS/JS)
  lib-node/           ← Node-only code (updater, store, API re-export)
tests/                ← Jest unit + integration tests (~600 tests)
desktop/tests/        ← Jest desktop tests (~98 tests)
```

**Running tests:**
```bash
npm test                          # extension tests
npm test -- --coverage            # with coverage
cd desktop && npm test            # desktop tests
```

Live API tests in `tests/utilization.test.js` require a `.env` file at project root (gitignored):
```bash
OPENAI_API_KEY=sk-...
GOOGLE_API_KEY=AIza...
CLAUDE_API_KEY=sk-ant-...
```

## FAQ

**Which provider should I pick?**
Google Gemini — free tier, generous quota, fast. Start there.

**How much does it cost to use?**
Fractions of a cent per action. Fixing a paragraph is roughly $0.0003–$0.001. Most users spend under $1/month.

**Is my writing stored anywhere?**
No. All data stays local on your device. Your text is only ever sent to the AI provider you chose (using your own API key) for processing — it goes nowhere else. Thought Tidy has no backend, no database, and no servers. Nothing is stored by us.

**Does it work on every website?**
It injects into all pages. Sites with aggressive CSP may block the inline modal — the action still runs and the result is shown in the extension popup.

**What is "Idea → Prompt"?**
A clarifying action — you describe a vague idea and Thought Tidy asks you a few questions to sharpen it, then outputs a clean AI prompt ready to paste into any tool.

**What is "Sound Like Me"?**
A Pro profile system. Describe yourself — name, role, writing style, context. Every action uses your profile as context so output sounds like *you* wrote it.

**Can I use this without publishing to a store?**
Yes. Chrome: load `dist/chrome/` as unpacked (stays loaded). Firefox: load `dist/firefox/` as temporary add-on (removed on restart — or self-sign with [web-ext](https://github.com/mozilla/web-ext)).

**Multiple computers?**
Yes. Same email and license key — just re-enter it on each machine.

**Can I contribute?**
Yes — PRs, issues, and forks are welcome. Run `npm test` before submitting.
